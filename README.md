# Ecommerce Trading Platform

## Project Overview

The Ecommerce Trading Platform is a comprehensive solution for online buying and selling. It connects buyers and sellers in a user-friendly environment with a focus on efficiency and security.

## Features
- User registration and authentication
- Product listing and categorization
- Shopping cart functionality
- Secure payment integration
- Order tracking and management
- User profiles and account management
- Admin dashboard for product and user management

## Tech Stack
- **Frontend:** React.js, Redux, CSS
- **Backend:** Node.js, Express.js, MongoDB
- **Deployment:** AWS, Docker
- **Testing:** Jest, Mocha

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/Jotham-M/ecommerce-trading-platform.git
   cd ecommerce-trading-platform
   ```
2. Install dependencies:
   - For frontend:
     ```bash
     cd client
     npm install
     ```
   - For backend:
     ```bash
     cd server
     npm install
     ```
3. Run the application:
   - For frontend:
     ```bash
     cd client
     npm start
     ```
   - For backend:
     ```bash
     cd server
     npm start
     ```
